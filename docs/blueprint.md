# **App Name**: Code Muse

## Core Features:

- Code Generation: Generate code snippets based on natural language prompts using an AI language model.
- Language Selection: Select a language, and automatically generate relevant code based on the selected language. The tool automatically uses common conventions from the specified languages to give appropriate results
- Code Editor: Display generated code in an editable text area for review and modification.
- Copy to Clipboard: Button to copy generated code to the clipboard for easy integration.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and intelligence, aligning with the application's function in code generation.
- Background color: Light gray (#EEEEEE), a desaturated version of the primary color that ensures comfortable readability.
- Accent color: Soft violet (#9575CD), analogous to blue, to highlight key interactive elements such as buttons and prompts.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern, machined look suitable for code and interface elements.
- Code font: 'Source Code Pro' for displaying code snippets within the app.
- Use minimalistic line icons to represent different coding languages and actions, maintaining a clean and efficient design.
- Subtle animations during the code generation process to provide visual feedback to the user.
